# sweetspicygoddess.com

Static website for **Ki’Loni Lee** — authenticity, creativity, and giving back to the community.

## Local preview

Open `index.html` in your browser, or run a tiny local server:

```bash
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.

## Deploy on Netlify

- Publish directory: `.`
- Build command: *(none)*

This repo includes `netlify.toml` preconfigured for static hosting + basic security headers.

## Forms

- Newsletter form (homepage) uses Netlify Forms: `form name="newsletter"`
- Contact form uses Netlify Forms: `form name="contact"`
